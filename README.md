# pluvion
Pluvion é uma ferramenta computacional voltada ao cálculo hidrológico para projetos de drenagem, focada na análise de bacias hidrográficas e no cálculo de parâmetros como tempo de concentração, intensidade de chuva e vazão de projeto, com base em métodos consagrados da engenharia e arquitetura modular para evolução futura.
